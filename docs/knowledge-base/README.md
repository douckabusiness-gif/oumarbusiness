# Knowledge Base Agents - Oumar Business

Ce dossier contient les documents de base a importer dans la Knowledge Base des agents.

Objectif :
- donner aux agents une methode de travail claire
- fixer des delais standards realistes
- definir les regles pour les projets personnels et internes
- garder des reponses coherentes entre WhatsApp, email, CRM et Marketplace

Documents inclus :
1. `01-methode-de-travail-oumar-business.md`
2. `02-delais-standards-projets-clients.md`
3. `03-regles-projets-personnels-et-internes.md`
4. `04-guide-decision-agents.md`

Usage recommande :
- importer chaque document comme source `manual` ou `file`
- scope : `global`
- type : `process`, `policy`, `timeline`, `faq`

Ces documents servent de reference pour :
- l'agent autonome d'accueil
- l'agent commercial
- l'agent chef de projet
- l'agent billing
- les agents sourcing
